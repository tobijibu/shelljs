ln()

更新日: 2018.09.11

ln([options,] source, dest)

利用可能オプション

-s
シンボリックリンク。
-f
強制的に上書きします。

指定方法

ln('file', 'newlink');
ln('-sf', 'file', 'existing');

sourceをdestにリンクします。 既にdestが存在する場合は、-fオプションを使って強制的に上書きます。


使用例

'shell1.txt'を'shell3.txt'にリンクします。 'shell1.txt'を書き換えると'shell3.txt'も同様に書き換えられます。

JavaScript
var sh = require('shelljs');
sh.ln('shell1.txt', 'shell3.txt');
